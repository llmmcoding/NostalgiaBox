## 游戏一：魔塔·黄金大陆 🏰

**分类：** 魔塔（回合制策略RPG）
**难度：** 中等
**目标用户：** 15-35岁，经典游戏爱好者

### 一、核心玩法

**类型：** 经典魔塔（Floor-Roguelike）
玩家在每层楼中击杀怪物、收集钥匙、开宝藏箱，50层内到达顶层救出公主。

**战斗系统：** 回合制自动战斗
- 玩家攻击 = 基础攻击力 + 装备加成
- 怪物攻击 = 固定伤害 - 玩家防御力
- 玩家先攻（可设置）
- 每层最多挑战3次怪物

**数值体系：**
```
初始属性：HP=1000, 攻击=10, 防御=5, 金币=0, 钥匙=3
每层怪物强度 = 基础值 × (1 + 楼层 × 0.15)
每5层出现商店，可购买HP上限/攻击/防御
```

**资源管理：**
- 🔑 钥匙：开不同颜色门（红门需2把/蓝门1把/黄门1把）
- 💰 金币：击杀怪物获得，用于商店购买
- ❤️ 血瓶：每层固定刷新，+200 HP

**操作方式：** 方向键/WASD 移动，碰到怪物自动战斗

### 二、游戏流程

```
第1层 → 新手引导（自动导航）→ 学习移动/战斗/钥匙使用
第5层 → 第一次商店
第10层 → BOSS：红色史莱姆王
第20层 → 商店升级为高级商店
第30层 → BOSS：骷髅王
第40层 → 最终BOSS：暗黑龙
第50层 → 救出公主 → 通关
```

### 三、UI设计

**游戏界面：**
- 左上：楼层显示 (第X层)
- 右上：HP条 + 金币 + 钥匙数量
- 中间：Canvas 像素地图（每格16x16px）
- 下方：道具栏（血瓶、钥匙数量）
- 侧边：怪物图鉴（已击杀/未击杀）

**地图元素：**
| 元素 | 颜色 | 说明 |
|---|---|---|
| 玩家 | 🟦 蓝色 | 可移动 |
| 楼梯 | 🟨 黄色 | 通向下一层 |
| 怪物 | 🔴 红色 | 碰触即战斗 |
| 墙壁 | ⬛ 黑色 | 不可通过 |
| 门 | 🚪 | 需要钥匙 |
| 宝箱 | 📦 | 随机道具/金币 |

### 四、技术实现

- **引擎：** 原生 HTML5 Canvas
- **地图生成：** 预设关卡脚本 + 随机怪物位置
- **帧率：** 30fps（像素风格）
- **存储：** localStorage 存档

## 游戏二：像素勇者 ⚔️

**分类：** 像素RPG（俯视角）
**难度：** 中等
**目标用户：** RPG爱好者

### 一、核心玩法

**类型：** 俯视角像素ARPG
玩家控制像素勇者，在迷宫中探索、收集装备、升级技能、挑战BOSS。

**战斗系统：** 即时制 + 技能
- WASD 移动，鼠标点击释放技能
- 普通攻击：近战挥剑（无冷却）
- 技能1：火球术（3秒冷却）
- 技能2：闪电链（5秒冷却）
- 跳跃：穿越窄道

**成长体系：**
```
等级1 → 初始 HP=100, 攻击=10, 防御=5
每升1级：HP+20, 攻击+3, 防御+2, 获得1点技能点
经验值来源：击杀怪物、完成NPC任务
```

**装备系统：**
- 武器：剑（攻击+5~20）
- 护甲：胸甲（防御+3~15）
- 饰品：戒指（暴击率+5%~20%）
- 随机掉落于怪物或商店购买

### 二、游戏流程

```
序章 → 新手村（学习基本操作）
第一章 → 森林迷宫（3个BOSS）
第二章 → 火山洞穴（5个BOSS）
终章 → 巨龙巢穴（最终BOSS：巨龙）
通关条件：击杀最终BOSS
```

### 三、UI设计

- 左上：HP条 + MP条
- 右上：等级 + 金币
- 下方：技能栏（4个技能格）
- 侧边：装备栏（武器/护甲/饰品）
- 地图：Canvas 俯视角（角色16x16，视野200x200px）

### 四、技术实现

- **引擎：** 原生 HTML5 Canvas + ES6类
- **地图：** 预设Tile地图编辑器JSON格式
- **AI：** 怪物状态机（巡逻/追击/攻击）
- **存储：** localStorage

## 游戏三：跳跳像素人 🏃

**分类：** 无限跑酷
**难度：** 简单
**目标用户：** 休闲玩家

### 一、核心玩法

**类型：** 横版无尽跑酷
玩家控制像素小人自动向右跑酷，单击/空格/上键跳跃，越过障碍物获得分数。

**操作：**
- 单击空格/上/W/点击屏幕：跳跃（可二段跳）
- 跳跃高度：150px，持续时间0.6秒
- 二段跳：在空中再跳一次，高度减半

**障碍物类型：**
| 障碍物 | 高度 | 宽度 | 出现时间 |
|---|---|---|---|
| 地洞 | 1格 | 2格 | 5秒后 |
| 石块 | 1格 | 1格 | 8秒后 |
| 飞来鸟 | 空中1.5格 | 1格 | 15秒后 |
| 高墙 | 2格 | 1格 | 20秒后 |

**分数系统：**
- 存活每秒：+10分
- 越过障碍物：+50分
- 二段跳：+20分
- 连续无伤10个障碍：+200分奖励

### 二、游戏流程

```
开始界面 → 点击开始 → 跑酷 → 撞到障碍 → GameOver → 显示分数 → 排行榜
```

### 三、UI设计

- 上方居中：当前分数（大字体）
- 左上：最高分
- 右：暂停按钮
- 角色：32x32像素奔跑动画（4帧循环）

### 四、技术实现

- **引擎：** 原生 HTML5 Canvas
- **物理：** 自定义重力模拟（重力加速度9.8，跳跃初速度15）
- **障碍生成：** 随机定时器（3-8秒间隔）
- **最高分：** localStorage

## 游戏四：宠物小精灵 🧩

**分类：** 休闲益智（消除+养成）
**难度：** 简单

### 一、核心玩法

**类型：** 三消（Puzzle Match-3）+ 宠物养成

**三消规则：**
- 6x8格盘面
- 交换相邻两个元素，横/竖3个相同即消除
- 消除得分，召唤精灵球
- 精灵球集满5个，捕捉随机精灵

**精灵系统：**
- 每局游戏随机出现 3 种精灵可捕捉
- 捕捉后精灵进入图鉴
- 图鉴收集解锁成就系统

### 二、UI设计

- 游戏区：Canvas（6x8格，每格48x48px）
- 下方：技能道具栏
- 右上：精灵球计数
- 右上角：图鉴入口

### 三、技术实现

- **引擎：** 原生 HTML5 Canvas
- **消除算法：** BFS Flood-Fill
- **动画：** CSS transitions

## 游戏五：魔塔·暗黑森林 🌲

**分类：** 魔塔（高难度版）
**难度：** 困难

### 一、核心玩法

在黄金大陆基础上：
- 地图更大（15x15）
- 怪物攻击力更高（+50%）
- 资源更稀缺（钥匙更少）
- 有debuff系统（中毒/冰冻/灼烧）
- BOSS有特殊机制

**新增系统：**
```
Debuff:
- 中毒：每回合 HP-50，持续3回合
- 冰冻：本回合无法行动
- 灼烧：HP-30/秒，持续2秒
```

### 二、BOSS设计

| BOSS | 楼层 | HP | 特殊机制 |
|---|---|---|---|
| 狼王 | 10层 | 3000 | 召唤2只小狼助阵 |
| 巨石傀儡 | 20层 | 5000 | 会击退玩家 |
| 巫妖王 | 40层 | 8000 | 全体debuff攻击 |

## 游戏六：龙之谷 🐉

**分类：** 像素RPG（史诗版）
**难度：** 困难

### 一、核心玩法

**类型：** 横版动作RPG（类似恶魔城）

- 6个关卡，每个关卡3小节
- 每个BOSS有独特机制
- 可收集4种元素碎片召唤神龙

**元素系统：**
- 🔥 火 + ❄️ 冰 + ⚡ 雷 + 🌿 自然
- 收集碎片后在篝火处融合，融合后召唤对应神龙助战

**BOSS列表：**
1. 哥布林王（关卡1）
2. 冰霜巨龙（关卡2）
3. 雷鸟（关卡3）
4. 自然守护者（关卡4）
5. 黑暗骑士（关卡5）
6. 最终巨龙（关卡6）

## 游戏七：方块跑酷 🟧

**分类：** 跑酷
**难度：** 简单

### 一、核心玩法

**类型：** 3D无尽跑酷（伪3D/2.5D）

与跳跳像素人的区别：
- 视角：2.5D等距视角
- 方向：可左右移动（←→键）+ 跳跃
- 障碍物：更丰富（旋转刀片、落石区、裂缝）
- 收集：方块金币（积累可解锁皮肤）

**金币系统：**
- 路上随机出现，收集得金币
- 金币可购买角色皮肤（5种）

## 游戏八：2048怀旧版 🔢

**分类：** 休闲益智
**难度：** 中等

### 一、核心玩法

**经典2048规则：**
- 4x4格盘面
- ←↑↓→滑动，所有方块向滑动方向合拢
- 相同数字相加（2+2=4, 4+4=8...）
- 达到2048算胜利，可继续挑战最大值

**怀旧元素：**
- 像素风格UI
- 8-bit 背景音乐
- 每次合成播放经典8-bit音效

**存档系统：**
- 最高分数 localStorage
- 可重置重新开始

### 二、技术实现

- **引擎：** 纯HTML5 + CSS动画
- **存储：** localStorage
- **音效：** Web Audio API (8-bit合成音)

## 技术统一规范

所有游戏必须满足：

| 规范 | 要求 |
|---|---|
| 帧率 | 稳定30fps以上 |
| 包体积 | 每个游戏 < 100KB |
| 启动时间 | < 2秒 |
| 存档 | localStorage，自动存档 |
| 断点续玩 | 支持 |
| 多语言 | 默认中文，可扩展 |
| 无网络依赖 | 纯本地运行 |
